# AI専門家コンサルタント

LangChainとStreamlitを使用した、複数の専門分野のAIエキスパートと対話できるWebアプリケーションです。

## 🎯 主な機能

- **5つの専門分野**: プログラミング、医療・健康、ビジネス・経営、教育・学習、クリエイティブ
- **専門家選択**: ラジオボタンで相談したい分野の専門家を選択
- **カスタマイズされた回答**: 各専門家に特化したシステムメッセージで最適な回答を提供
- **直感的なUI**: 使いやすいStreamlitインターフェース
- **リアルタイム相談**: 即座に専門的なアドバイスを取得

## 🚀 利用可能な専門家

### 1. プログラミング専門家
- プログラミング、ソフトウェア開発、技術的な問題解決に特化
- コード例を含む実用的なアドバイスを提供

### 2. 医療・健康専門家
- 医学、健康管理、栄養、フィットネスに関する知識提供
- 科学的根拠に基づいた情報を提供（診断・治療は医師への相談を推奨）

### 3. ビジネス・経営専門家
- 経営戦略、マーケティング、財務、人事、起業に関するアドバイス
- 実践的で戦略的な指導を具体例とともに提供

### 4. 教育・学習専門家
- 効果的な学習方法、教育技術、知識習得に関する指導
- 教育学的根拠に基づいた年齢・レベル別の提案

### 5. クリエイティブ専門家
- デザイン、アート、文章作成、音楽、映像制作のサポート
- 創造性を刺激する技術的アドバイスとツール使用方法

## 💻 セットアップ

### 1. 仮想環境のアクティベート（既に作成済みの場合）

```powershell
.\env\Scripts\Activate.ps1
```

### 2. 依存関係のインストール

```powershell
pip install -r requirements.txt
```

### 3. OpenAI APIキーの設定

`.env`ファイルを作成し、以下を追加してください：

```
OPENAI_API_KEY=your_openai_api_key_here
```

APIキーは[OpenAI Platform](https://platform.openai.com/api-keys)から取得できます。

### 4. アプリケーションの実行

```powershell
streamlit run app.py
```

## 📱 使用方法

1. **ブラウザでアクセス**: http://localhost:8501 を開く
2. **専門家を選択**: 左側のラジオボタンから相談したい分野を選択
3. **質問を入力**: 右側のテキストエリアに具体的な質問を入力
4. **回答を取得**: 「専門家に相談する」ボタンをクリック
5. **結果を確認**: 選択した専門家からの詳細な回答を確認

## 🌐 Streamlit Community Cloudへのデプロイ

このアプリケーションはStreamlit Community Cloudでのデプロイに対応しています：

- **Pythonバージョン**: 3.11（`.python-version`ファイルで指定）
- **環境変数**: Streamlit Cloud上でOPENAI_API_KEYを設定
- **依存関係**: `requirements.txt`に全て記載済み

### デプロイ手順

1. GitHubリポジトリにコードをプッシュ
2. [Streamlit Community Cloud](https://share.streamlit.io/)にアクセス
3. GitHubリポジトリを選択してデプロイ
4. 環境変数にOPENAI_API_KEYを設定

- **言語モデル**: OpenAI GPT-4o-mini

## 📝 注意事項

- 各専門家は一般的な知識に基づいてアドバイスを提供します
- 医療や法律に関する専門的な判断が必要な場合は、実際の専門家にご相談ください
- AI生成の内容のため、情報の正確性を確認することをお勧めします
- OpenAI APIの使用にはクレジットが必要です

## 🛠️ トラブルシューティング

### OpenAI APIキーエラー
- `.env`ファイルが作成されているか確認
- APIキーが正しく設定されているか確認
- APIキーに十分なクレジットがあるか確認

### インポートエラー
LangChainの最新バージョンを使用していることを確認：

```powershell
pip install --upgrade langchain langchain-openai langchain-core
```

### パッケージインポートエラー
必要なパッケージがインストールされているか確認：

```powershell
pip list
```
